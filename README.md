Dotfile management

Clone repo into ~/.config/docrepo (~/.config> git clone git@github.com:iveyj1/dotrepo.git)

Move aside existing dirs in ~/.config

Add symlink from each dir in docrepo/ to  ~/.config/ 

To add, delete, or edit a file in an existing dir, just add/delete/edit it as normal

To add a new top level directory, move the dir to docrepo/, then add a symlink

Manage changes with git operations on dotrepo/
